# fav_explosion
## 利用方法
- git clone
```
$git clone git@github.com:wheson/fav_explosion.git
```
- gem install
```
$gem install twitter
$gem install dotenv
```

- .envファイルを下記のように記入し，fav_explosionディレクトリ上に置く
```
# それぞれ取得したものを入れる
CONSUMER_KEY=""
CONSUMER_SECRET=""
ACCESS_TOKEN=""
ACCESS_TOKEN_SECRET=""
```

- 起動する
```
$ruby main.rb 
```
